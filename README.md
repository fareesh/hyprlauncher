## hyprlauncher
A multipurpose and versatile launcher / picker for Hyprland

![](./assets/preview.png)

## Features

- Various providers: Desktop, Unicode, Emoji, Math ...
- Fast, multi-threaded fuzzy searching
- Daemon by default
- Very fast opening

## Runtime dependencies

- Desktop: none
- Unicode: `wl-copy`
- Math: `wl-copy` for copying the result
